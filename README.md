### 如何使用

本地使用只需下载本仓库并使用Obsidian打开即可。

#### Git 快捷键

1. Commit all: `Alt + C`
2. Pull: `Ctrl + Shift + -`
3. Push: `Ctrl + Shift + +`

> 你可以在设置中更改自己的快捷键

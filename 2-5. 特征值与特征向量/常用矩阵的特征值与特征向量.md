#重要 

![[eigen_form.png]]

后四列为重点内容，请一定记忆！

> [!example] 
> 设矩阵 $A$ 满足 $A^{2}=A$ ，求其特征值可能的取值。
> 
> $$A^{2}-A = 0 ~\Rightarrow~ f(A) = 0 ~\Rightarrow~ f(\lambda) = 0 ~\Rightarrow~ \lambda^{2} - \lambda = 0$$
> $$\lambda (\lambda -1) = 0 ~ \Rightarrow ~ \lambda = \{ 0,1 \}$$

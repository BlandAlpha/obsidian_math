$$\sec x$$
1. $\sec x = \frac{1}{\cos x}$
2. 偶函数
3. 导数：

$$\csc x$$
1. $\csc x = \frac{1}{\sin x}$
2. 奇函数
3. 导数

$$\ln (x + \sqrt{1+x^2}) $$
1. 奇函数
2. 导数：$\frac{1}{\sqrt{1+x^2}}$
3. 当 $x \to 0$，$\ln (x + \sqrt{1+x^2}) \sim x$

$$[x]$$
1. 取整函数
2. 本质： $x-1 < [x] \le x$ （可用于夹逼）
